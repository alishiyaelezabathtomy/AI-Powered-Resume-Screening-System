# AI-Powered Resume Screening System

## Overview

An AI-based resume screening application that automates candidate evaluation and job-role matching using Natural Language Processing and Machine Learning.

## Objective

The system helps reduce manual resume screening effort by analyzing resumes and matching candidates with suitable job requirements.

## Technologies Used

- Python
- Natural Language Processing (NLP)
- Scikit-learn
- Pandas
- NumPy
- SQLite
- MySQL

## Features

- Resume text extraction
- Text preprocessing
- Feature extraction
- Candidate-job matching
- Resume ranking
- Candidate information management

## Machine Learning Approach

- Data preprocessing
- NLP techniques
- Feature engineering
- Classification models

## Database

SQLite/MySQL is used to store and manage candidate information.

## Future Enhancements

- Resume parser improvement
- Advanced AI-based ranking
- Web deployment using Django
